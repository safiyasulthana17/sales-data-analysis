# 📊 Sales Data Analysis Project

## 📌 Project Overview
This project analyzes sales data to uncover trends, top-performing products, and high-revenue regions.  
It also includes sales forecasting and a Power BI dashboard for business decision-making.

## 🛠 Tools & Technologies
- Python (Pandas, NumPy, Scikit-learn)
- SQL-style queries (pandasql)
- Power BI
- Data Visualization
- Machine Learning (Linear Regression)
